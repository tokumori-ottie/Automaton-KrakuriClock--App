# 🕰️ からくりとけいメーカー

5〜7歳の子どもが一人で楽しめる「からくり時計づくり」体験PWA。
6ステップのウィザードで時計を組み立てると、毎時0分にからくりアニメーションが動く。

## 特徴

- **ゼロ依存**：HTML / CSS / JS のシングルファイル構成（`index.html` のみ）
- **音声合成**：Web Audio API によるプログラム生成BGM（著作権フリー・音声ファイルなし）
- **PWA対応**：ホーム画面追加・オフライン動作
- **フォト対応**：カメラロールから文字盤・キャラクター写真を取り込み可能
- **実績システム**：作成数に応じて隠しキャラクターが解放される

## ウィザードステップ

| Step | 内容 |
|------|------|
| 1 | 時計の名前（最大12文字） |
| 2 | 時計の形（まる / パネル / しかく / アーチ） |
| 3 | テーマカラー（文字盤色・わく色 / 写真選択可） |
| 4 | 針の色・数字スタイル・数字の色 |
| 5 | キャラクター・アニメーション種類（複数選択可） |
| 6 | からくり音楽（10曲から選択） |

## ローカル起動

```bash
python3 -m http.server 8080
# → http://localhost:8080（モバイルビュー推奨・portrait 390px）
```

## ファイル構成

```
index.html      # アプリ本体（HTML + CSS + JS 埋め込み・約4,350行）
sw.js           # Service Worker（キャッシュ名: karakuri-v1）
manifest.json   # PWA設定
icon-192.png    # アプリアイコン
icon-512.png    # アプリアイコン
REQUIREMENTS.md # 要求仕様書（日本語）
CLAUDE.md       # AI開発ガイド
```
