# 🕰️ からくりとけいメーカー 要求仕様書

バージョン 1.1 | 2026年3月

---

## 1. プロジェクト概要

### 1.1 目的

5〜7歳の子どもが一人で楽しめる「からくり時計づくり」体験を提供するPWA（Progressive Web App）。
子どもが色・キャラクター・音楽などを自分で選びながら時計を組み立て、毎時ちょうどになるとからくりが動くインタラクティブな知育アプリ。

### 1.2 ターゲットユーザー

- 主ユーザー：5〜7歳の子ども
- 保護者（セットアップ・GitHub Pages公開を行う）
- 将来的にはGoogle Play / App Store での一般公開も視野

### 1.3 技術スタック

- フロントエンド：HTML / CSS / JavaScript（シングルファイル構成）
- ホスティング：GitHub Pages
- 音声：Web Audio API（著作権フリーのプログラム生成メロディ）
- グラフィック：Canvas API + CSS/DOM ハイブリッド
- データ保存：localStorage（最大5件）
- PWA：manifest.json + Service Worker（オフライン対応）

## 3. ウィザード仕様（6ステップ）

各ステップの選択後にインタラクティブな製作演出が挿入される。

| Step | タイトル | 選択内容 | 製作演出（インタラクティブ） |
|------|----------|----------|------------------------------|
| 1 | なまえ | 時計の名前（最大12文字・プリセット3種＋自由入力） | なし |
| 2 | とけいのかたち | 本体形状4種（まる/パネル/しかく/アーチ） | 🧱 ブロックをはめる（1回） |
| 3 | テーマカラー | 文字盤色・わく色（同時）/ カメラロール写真選択可 | 🖌️ 指でなぞって塗る（SVGペイント） |
| 4 | はりとすうじ | 針の色・数字スタイル・数字の色（同時） | 🔨 タップしてハンマーで釘を打つ（5回） |
| 5 | キャラとうごき | キャラクター・アニメーション種類（複数可）/ カメラロール対応 | 👆 タップしてキャラを設置（3回） |
| 6 | おんがく | からくり起動時の音楽（10曲から選択） | ⚙️ タップしてぜんまいを巻く（3回） |

### 3.1 ウィザードUI仕様

- ヘッダー右端に「組み立て中ミニプレビュー」（52×52px）を常時表示。ステップが進むにつれ時計が段階的に完成していく
- 「つぎへ すすむ」ボタンは画面下部に固定表示（font-size: 1.5rem、padding: 22px）
- 色ドットは44px以上（5歳児でもタップしやすいサイズ）
- ステップ3・4のプレビューcanvasは220×220px（リアルタイム針更新付き）
- 製作演出中は「あと○かい！」を3.5remの大きな文字で残り回数表示
- プログレスバーで達成度を可視化
- **選択フィードバック**：選択肢タップ時に機械音（Web Audio）＋プレビューcanvasフラッシュ/バウンスアニメ

### 3.2 カメラロール・トリミング

- 文字盤（ステップ3）・キャラクター（ステップ5）の両方でカメラロールから写真選択可能
- 写真選択後、丸くトリミングするUIが起動（ドラッグで位置調整・スライダーで拡大縮小）

### 3.3 時計の形（CLOCK_BODIES）

| ID | 名前 | 説明 |
|----|------|------|
| round | まる | シンプルなまるい時計 |
| panel | パネル | 4まいのパネルがひらく |
| square | しかく | かどのあるしかく時計 |
| arch | アーチ | ふりこのついた時計 |

## 4. からくりアニメーション仕様

### 4.1 起動条件

- 毎時0分に自動起動（リアルタイムモード時）
- 「テストする」ボタンで手動起動
- 5.5秒後に自動終了

### 4.2 キャラクターアニメーション種類（7種類）

| ID | 名前 | 内容 |
|----|------|------|
| dance | ダンス | 左右に揺れながら回転 |
| bounce | ジャンプ | 上下に跳ねる |
| spin | くるくる | 360度回転し続ける |
| wave | ふりふり | 手を振るように揺れる |
| split | 4ぶんかつ | 文字盤が4扇形に分割され、各扇が独立して自転（セイコーからくり時計参考） |
| door | とびら | Canvas上で4枚の扇形扉が内側に折れて開き、中央からキャラが登場。⭐が周回する |
| hanabira | はなびら | 6枚の花びら状扇形が交互に回転して開き、キャラが登場（セイコー実機風） |

複数選択可能。split / door / hanabira が複数選ばれた場合はランダムで切り替わる。

### 4.3 小屋（からくり装置）

- 時計の上部に設置される小屋（houseBody）から二枚扉が開いてキャラが飛び出す（デフォルト演出）
- わくの色に連動して小屋の色が変わる

## 6. 出荷シーケンス（完成フロー）

ウィザード最終ステップで「かんせい！」を押すと3フェーズの演出が始まる。

1. **フェーズ1：カウントダウン（3→2→1）**
   - ドキドキ感を演出するサウンド付き
   - 暗い背景に大きな黄色の数字

2. **フェーズ2：最終チェックリスト**
   - 7項目が1つずつ✅になる（カチカチ音付き）
   - もじばんの いろ / はりの いろ / わくの いろ / すうじの スタイル / キャラクター / うごきかた / おんがく

3. **フェーズ3：連続タップで出荷**
   - 「タップして おくりだそう！」
   - 10回タップで📦が画面外に飛び立つ
   - ファンファーレ音楽

4. **完成画面（pgDone）**
   - 時計がくるくる回転しながら登場（CSSアニメーション）
   - 「○○が つくったよ！🎊」と名前入り呼びかけ
   - 作った時計が実際に動く（リアルタイム針）
   - 3連続紙吹雪

## 8. データ仕様

### 8.1 保存データ構造（localStorage）

| キー | 内容 | 形式 |
|------|------|------|
| `kk_clocks` | 保存した時計一覧（最大5件） | JSON配列 |
| `kk_stats` | 作成数・アンロック済みキャラ | JSON オブジェクト |
| `kk_meta` | メタデータ（初回起動日など） | JSON オブジェクト |

### 8.2 時計データフィールド（kk_clocks の各要素）

| フィールド | 型 | 説明 |
|------------|-----|------|
| id | number | Date.now()で生成する一意ID |
| name | string | 時計の名前（最大12文字） |
| clockBody | string | 本体形状（"round" \| "panel" \| "square" \| "arch"） |
| faceColor | string | 文字盤の色（HEX） |
| faceImage | string \| null | 文字盤の写真（base64 DataURL） |
| frameColor | string | わくの色（HEX） |
| handColor | string | 針の色（HEX） |
| numColor | string | 数字の色（HEX） |
| numberStyle | string | "arabic" \| "roman" \| "dots" \| "none" |
| character | string | キャラクター絵文字 |
| charImage | string \| null | キャラクター写真（base64 DataURL） |
| anims | string[] | アニメーションID配列（複数選択可） |
| music | string | 音楽ID（デフォルト："chime"） |

## 9. UI/UXガイドライン

### 9.1 デザイン原則

- ターゲットは5歳児：文字は少なく・絵文字とビジュアルで伝える
- タップ対象は最低44px（推奨64px以上）
- 「次に何をすればいいか」が瞬時にわかるUI（固定フッターボタン + 残り回数表示）
- 選択肢を選ぶたびに即座にプレビューが変わる（リアルタイムフィードバック）
- 「作ってる感」を出すインタラクティブ演出（なぞる・タップ・ドラッグ）
- 選択時の機械音フィードバック（Web Audio API）でからくり感を演出

### 9.2 カラーパレット

| 変数名 | HEX | 用途 |
|--------|-----|------|
| --red | #FF6B6B | メインカラー・CTAボタン |
| --yellow | #FFD93D | アクセント・テキストシャドウ |
| --green | #6BCB77 | 完了・成功 |
| --blue | #4D96FF | 情報・ナビゲーション |
| --purple | #C77DFF | アクセント |
| --orange | #FF9A3C | グラデーション補助色 |
| --bg | #FFF8F0 | 背景色（温かみのあるオフホワイト） |

### 9.3 フォント

- 見出し：Fredoka One（丸みのあるかわいいフォント）
- 本文：Nunito（読みやすい丸ゴシック系）

### 9.4 ページ遷移

- 左右スライドアニメーション（cubic-bezier(.4,0,.2,1) 380ms）
- 進む方向：右からスライドイン
- 戻る方向：左からスライドイン
